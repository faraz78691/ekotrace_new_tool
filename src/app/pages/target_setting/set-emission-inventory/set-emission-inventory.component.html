Set Emission Inventory

EKO Trace aligns with science-based target initiative (SBTi) to set the targets for emission reductions. SBTs are in line with the scale of reductions that the latest climate science deems necessary to keep global warming below 1.5°C. The goal is to reduce global GHG emissions by 50% by 2030 and reach net zero emissions by no later than 2050. Setting SBTs in EKO Trace is a three-step process:

Set Emissions Inventory: An annual emissions inventory is a prerequisite for emissions forecasting and setting targets. The target setting can only be done for the parent company.

  1. Entering the inventory data
    • Inputting scope1, scope2 and scope3 data for the corresponding year is mandatory. Subcategory emissions data is optional and is required only when you are selecting other target KPIs.
    • Product output is required for target type ‘physical intensity convergence.’ This target type is useful in the case of single product manufacturing. The unit of product output can be no. of items / tonnes / kg etc.
    • Economic output is required for ‘economic intensity convergence’ and is useful when the company generates revenue from multiple products. The unit of economic output is the local ccy of revenue output
  2. Emissions growth, revenue and output growth projections are required for emissions forecasting. To do the forecasting, you at-least need the base year data (the year when target setting was done), current year (which has latest emissions data) and the target year (by when you want to achieve the target reduction in emissions).
